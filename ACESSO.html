Como Entrar na App

Email de admin
O email que usaste para criar a primeira conta
Palavra-passe
Escreve aqui a tua palavra-passe num local seguro
Esqueci a palavra-passe No ecrã de login → separador "Esqueci" → escreve o email → clica "Enviar link"
Recebes um email da Google para repor a palavra-passe

Primeira Vez (criar conta de admin)

Abre https://serralves-decor.web.app

Clica no separador "Criar conta"

Escreve o teu email e uma palavra-passe (mínimo 8 caracteres)

Clica "Criar conta" — a primeira conta criada é automaticamente administradora

Para adicionar colegas: vai a Equipa → Adicionar novo membro (só o admin vê este separador)

Papéis na App

Administrador
  • Criar orçamentos
  • Ver histórico
  • Editar produtos e tecidos
  • Alterar configurações
  • Gerir equipa (convidar / remover)
  • Exportar / importar dados
  • Mudar funções da equipa
Utilizador
  • Criar orçamentos
  • Ver histórico
  • Ver catálogo (só leitura)
  • Ver configurações (só leitura)
  • Não vê separador Equipa
  • Não pode alterar dados

Problemas Frequentes

Esqueci a palavra-passe Login → "Esqueci" → enviar link para o email de admin
App não abre Tenta a URL alternativa: https://espacoprovavel.github.io/appserralves/app-orcamentos/
Dados desapareceram Abre noutro dispositivo — os dados estão na nuvem (Firestore). Se apagados por engano, contacta suporte técnico.
Colega não consegue entrar Admin → separador Equipa → verifica se a conta está activa
PDF sai em branco Usa o botão "Imprimir" em vez de "Descarregar PDF" (fallback via browser)
Deploy falhou (fundo vermelho no GitHub Actions) Verifica se o secret FIREBASE_SERVICE_ACCOUNT_SERRALVES_DECOR existe em GitHub → Settings → Secrets
